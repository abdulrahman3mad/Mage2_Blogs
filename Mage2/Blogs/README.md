# Mage2 Blogs Module

    mage2/module-blogs

### Main Functionalities

* ***Show*** blogs in admin grid
* ***Add*** blogs using admin forms
* ***Import*** blogs from a third party API, and creates a ***BlogImporter*** that can be triggered
  from a button shown on store configuration, or via CLI
* ***CLI*** for (adding blogs, removing blogs, clearing all blogs, showing all blogs, showing a specific blog, updating
  a blog, importing blogs)
* ***Create*** blogs list [PLP]
* ***Create*** a blog page to show a blog details [PDP]
* ***Restful API end-points*** for (adding a blog, removing a blog, fetching all blogs, updating a product)
* ***GraphQL endpoints*** for (adding a blog, removing a blog, fetching all blogs, updating a product)
* ***Config*** the extension using store configuration (API key, API Url, Enabling/Disabling)
* Unit testing
* Add ***API service layer***
* Export blog in json/xml formats
* Add blogs using data patches
* Add translation to Arabic file
